# Shop Accounting System

## Description
A simple shop accounting system developed using Python, Flask, and SQLite. The application helps manage customers, orders, and financial records through a web interface.

## Features
- Add, edit, and delete customers
- Create and manage orders
- Search for customers
- Store data using SQLite
- Web interface built with HTML templates

## Technologies
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run

1. Install Flask:
```bash
pip install flask
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and go to:

```text
http://127.0.0.1:5000
```
